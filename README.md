# FinFlow — трекер доходов и расходов

Веб-приложение для учёта личных финансов с аналитикой по категориям и встроенным финансовым ассистентом.

**Ссылка на приложение:**
**Тестовый аккаунт:** demo@пgmail.com / demo13

## Возможности

- Регистрация и вход (email/password + Google)
- Учёт доходов и расходов по категориям
- Дашборд: баланс за месяц, диаграмма расходов, превышение лимитов по категориям
- Фильтрация операций по типу и категории
- Просмотр/редактирование/удаление отдельной операции
- Чат-ассистент: отвечает на вопросы про баланс, расходы по категории, лимиты
- Адаптивная вёрстка для мобилки

## Стек

React 18, React Router, Firebase (Authentication + Firestore+Analitics), Recharts, Vite, ESLint, GitHub
