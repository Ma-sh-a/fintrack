# FinFlow — трекер доходов и расходов

Веб-приложение для учёта личных финансов: доходы и расходы по категориям, накопления, инвестиционный портфель и финансовый ассистент — в одном месте.

**Ссылка на приложение:** https://fintrack-5wlb.vercel.app/dashboard
**Тестовый аккаунт:** demo@fintrack.test / demo1111
*(тестовый аккаунт заранее наполнен данными)*

## Возможности
- Регистрация и вход (email/password + Google)
- Учёт доходов и расходов по категориям — отдельные категории для доходов и расходов
- Дашборд: баланс за месяц с переключением между месяцами, диаграмма расходов по категориям, динамика доходов/расходов за полгода, превышения лимитов бюджета
- Фильтрация операций по типу и категории
- Просмотр/редактирование/удаление отдельной операции
- Копилки — цели с прогресс-баром и историей пополнений
- Инвестиционный портфель — учёт активов по брокерам и типам, расчёт доходности, автообновление котировок по тикеру через публичные API,расчёт риск-профиля портфеля
- Чат-ассистент
- Имитация подключения банков

**Реализовано**
- Адаптивная вёрстка для мобильных устройств
- Обработка ошибок сети (баннеры с понятными сообщениями вместо пустых экранов)
- Валидация форм
- Юнит-тесты бизнес-логики (Vitest)
- Мониторинг ошибок в проде (Sentry)
- CI/CD: линтинг, проверка форматирования (Prettier), тесты и сборка

## Стек
React 18, React Router, Firebase (Authentication + Firestore), Recharts, Vite, ESLint + Prettier, Vitest, Sentry, Vercel.

## Запуск локально

npm run dev       # локальный сервер разработки
npm run test      # юнит-тесты
npm run lint      # проверка кода
npm run format    # автоформатирование
npm run build     # продакшен-сборка


## Деплой
Деплой настроен автоматически через **GitHub Actions**: при пуше в main прогоняются линтинг, проверка форматирования, тесты и сборка, после чего результат публикуется на Vercel.
